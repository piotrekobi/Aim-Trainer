union-value

‍Set an array of unique values as the property of an object. Supports setting deeply nested properties using using object-paths/dot notation.

Install

Install with npm:

$ npm install --save union-value

Usage

var union = require('union-value');
var obj = {};
union(obj, 'a.b.c', ['one', 'two']);
union(obj, 'a.b.c', ['three']);
console.log(obj);
//=> {a: {b: {c: [ 'one', 'two', 'three' ] }}}
